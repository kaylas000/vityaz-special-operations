## 🔧 Интеграция в Код

### AssetManager.ts - Управление всеми ассетами

```typescript
import AssetManager from '../managers/AssetManager';

// Получить анимацию персонажа
const walkAnim = AssetManager.getCharacterAnimation('walk');
// → { name: 'walk', frames: [...], loop: true }

// Получить врага
const fighter = AssetManager.getEnemyType(0);
// → { type: 0, name: 'Fighter', hp: 30, damage: 8, speed: 1.0 }

// Получить эффект
const explosion = AssetManager.getEffect('explosion');
// → { name: 'explosion', frames: [...], loop: false }

// Получить оружие
const ak74m = AssetManager.getWeapon(0);
// → { id: 0, name: 'AK-74M', damage: 15, magazine: 30, recoil: 2 }
```

## 🎨 Технология

### Как это работает:

1. **Процедурная генерация** - Все спрайты созданы алгоритмами
2. **PIL (Python Imaging)** - Используется для создания PNG файлов
3. **Математические функции** - sin/cos для волновой анимации
4. **Параметрические модели** - Легко масштабировать и менять
5. **TypeScript типизация** - Полная безопасность типов
